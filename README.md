# Pembangunan Website Open Trip Wisata ke Banda Neira

Website Open Trip wisata ke Banda Neira dikembangkan sebagai media informasi dan pemesanan perjalanan wisata secara daring. Sistem ini menyediakan fitur registrasi dan login bagi pengguna untuk melakukan booking, pembayaran, melihat pesanan, serta mengajukan reschedule perjalanan.

Selain pengguna, sistem juga memiliki admin yang berperan dalam mengelola data paket wisata, jadwal perjalanan, data pengguna, serta memverifikasi pembayaran dan mengelola pesanan yang masuk. Admin juga dapat melakukan pembaruan informasi pada website guna memastikan data yang ditampilkan selalu akurat dan terkini.
